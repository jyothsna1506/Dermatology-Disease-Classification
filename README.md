# Dermatology-Disease-Classification
🧠 Dermatology Disease Classification using Machine Learning

This project implements a Machine Learning model to classify dermatological diseases using clinical attributes from a dermatology dataset. The aim is to analyze medical features and predict the correct disease category using supervised learning techniques.
📌 Project Overview

Dermatological diseases often have similar symptoms, which can make diagnosis difficult. This project demonstrates how machine learning can assist in disease classification by analyzing patient attributes.

The project includes:

Data preprocessing

Exploratory Data Analysis (EDA)

Machine Learning model training

Model evaluation and prediction

📊 Dataset

The project uses a Dermatology dataset containing multiple clinical attributes related to skin diseases.

Dataset features include:

Clinical attributes of dermatological conditions

Multiple disease classes for classification

Numerical values representing medical characteristics

🛠️ Technologies Used

Python

Jupyter Notebook

Pandas

NumPy

Seaborn

Scikit-learn

🤖 Machine Learning Model

The following model is implemented in this project:

Gaussian Naive Bayes

Steps followed:

Import and load dataset

Data preprocessing and cleaning

Feature selection and target variable separation

Train-test data splitting

Model training using Gaussian Naive Bayes

Prediction and performance evaluation
